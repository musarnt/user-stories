# Week 2 - User Story: Corporate Talent Hub

## Objective
Evolve the system to allow dynamic interaction and processing of multiple employees.
Implementing control flow structures and data handling by comparing Legacy Java 8 syntax
directly with modern LTS features (Java 17/21), ensuring system robustness through exception handling.

## Tasks
- **Task 1** - Switch Legacy vs Modern: main menu with classic switch and salary categorization with switch expression
- **Task 2** - Dynamic input and type inference: do-while loop with Scanner and var keyword
- **Task 3** - Performance matrix and casting: double[][] matrix, nested loops, and explicit casting
- **Task 4** - Robustness and error diagnosis: try-catch, InputMismatchException, and ternary operator

## Files
| File | Description |
|---|---|
| `Main.java` | Main entry point — integrates all tasks |
| `MainMenu.java` | Legacy switch menu and modern switch expression for salary categorization |
| `PerformanceMatrix.java` | Performance matrix with nested loops and explicit casting |
| `RobustSystem.java` | Exception handling, input validation, and promotion evaluation |
