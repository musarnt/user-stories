# Week 1 - User Story: Corporate Talent Hub

## Objective
Build the technical foundation of the "Corporate Talent Hub" system.
Modeling core data structures and implementing business logic using Java primitives,
modern Java 17/21 features, and operator precedence rules.

## Tasks
- **Task 1** - Architecture & Java ecosystem notes (Legacy vs Modern LTS)
- **Task 2** - Data modeling: traditional class vs modern Record
- **Task 3** - Business logic: operator precedence and salary calculation
- **Task 4** - Entry point: null handling and object reference comparison

## Files
| File | Description |
|---|---|
| `ArchitectureNotes.java` | JVM, GC, and Java 8 vs 17/21 documentation |
| `Employee.java` | Traditional class with all 8 primitive types |
| `CompanyRecord.java` | Modern Record with company data |
| `Main.java` | Main entry point — Tasks 3 and 4 |

Java 17+
